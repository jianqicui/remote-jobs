# Software Engineer – Python & Kubernetes (Remote, Worldwide)

## About the Role
Canonical is building a new Python-based open source operations framework for applications running on Kubernetes. You’ll help design, build, and maintain these systems across the Linux stack. The goal is to improve reliability, automation, and scalability for both internal and customer workloads.

## What You’ll Do
- Develop open source operations code in Python  
- Work across kernel, networking, storage, and application layers  
- Build reliable, testable, well-designed software  
- Contribute to monitoring, alerting, and automation  
- Simplify operations for users and the open source community  
- Collaborate using agile practices  
- Mentor teammates and support a healthy engineering culture  
- Work remotely and travel globally 1–2 times per year  

## What We’re Looking For
- Strong Python development skills  
- Experience with Linux system administration  
- Familiarity with Docker, Kubernetes, and at least one public cloud  
- Understanding of reliable software practices  
- Solid organizational and communication abilities  
- Degree in CS, STEM, or equivalent experience  
- Passion for technology and distributed systems  

## What Canonical Offers
- Fully remote work with periodic in-person team events  
- USD 2,000 annual learning budget  
- Annual compensation review and performance bonuses  
- Global, fair benefits including leave, parental support, and EAP  
- Travel opportunities and upgrades for long-haul events  

## About Canonical
Canonical is the company behind Ubuntu and a leader in open source, cloud, AI, and IoT. It maintains a high hiring bar and embraces diversity, inclusion, and remote-first work.
