# Senior Software Engineer (Tech Lead, React/Next/Nest)
**Remote | Full Time**

RV LIFE is seeking a senior engineer to lead development of scalable React, React Native, and AWS serverless applications.

## Responsibilities
- Lead architecture and development of React/React Native apps.
- Build greenfield applications from scratch.
- Explore and adopt new technologies.
- Design AWS serverless systems using Lambda, API Gateway, DynamoDB, S3.
- Build and maintain backend services and APIs.
- Provide technical leadership and solve complex problems.
- Develop and integrate features for web and mobile apps.
- Influence technical direction and best practices.
- Work independently or collaboratively as needed.
- Improve agile processes and CI/CD pipelines.
- Proactively identify and resolve issues.
- Mentor teammates and share knowledge.
- Conduct code reviews and maintain high standards.

## Minimum Experience
- 10+ years software development, 6+ years building scalable React apps.
- Strong React Native experience, including from-scratch projects.
- Hands-on AWS serverless experience.
- BS/MS in CS or equivalent experience.
- Strong understanding of cloud architecture, agile methodologies, and security best practices.

## Preferred Experience
- AWS Solutions Architect or equivalent experience.
- Strong TypeScript experience.
- Automated testing with Jest, Cypress.
- Micro frontends experience.
- Solid design patterns knowledge.
- Experience with location-based services (Google Maps, HERE).
- Responsive design and Atomic Design.

## Bonus
- Passion for RVing.
- Strong product ownership.
- Strong analytical and strategic thinking.

## Company Culture
- Innovation, collaboration, continuous learning.
- Fully remote and supportive work environment.

## Benefits
- Competitive salary.
- Flexible working hours.
- Professional development opportunities.
- Remote work from anywhere.

## Application Process
- **Submit a detailed application**, including:
  - Experience with greenfield projects
  - Examples of AWS serverless work
  - How you stay up to date with new technologies

  - Examples of independent and collaborative work
  - Current resume
